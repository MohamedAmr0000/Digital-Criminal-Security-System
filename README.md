# 🛡️ Aura-Guard: AI-Powered Digital Forensic & Deception Detection System

An advanced **Digital Forensic** web platform designed to analyze human truthfulness by integrating **Biometric Data** and **Artificial Intelligence**.

---

## 📝 Project Overview
The system monitors physiological and behavioral changes in real-time to provide high-accuracy deception detection for investigative purposes. It bridges the gap between hardware sensors, secure backend processing, and AI behavioral analysis.

## 🚀 Key Features
* **AI Facial Expression Analysis:** Uses computer vision to detect micro-expressions and facial cues associated with stress or deception.
* **Real-time Biometric Monitoring:** Integrates with hardware (**MAX30102 sensor**) to track heart rate and oxygen saturation (SpO2) fluctuations.
* **Secure Data Logging:** Built with a robust backend to store and analyze forensic sessions securely.
* **Interactive Dashboard:** A professional interface to visualize heart rate charts and AI analysis results simultaneously.

---

## 🛠️ Tech Stack
* **Frontend:** React.js / HTML5 / CSS3 (Modern & Responsive UI).
* **Backend:** Java / Spring Boot (Scalable & Secure API).
* **Database:** Oracle Database (Relational Data Integrity).
* **Hardware:** ESP32 Microcontroller & MAX30102 Heart Rate Sensor.
* **AI/ML:** Python (OpenCV/TensorFlow) for facial recognition and analysis.

## 📂 Project Modules
1. **Hardware (ESP32):** Reads data from MAX30102 and transmits via WebSockets/HTTP.
2. **Backend (Spring Boot):** Manages users, forensic data, and business logic.
3. **AI Engine (Python):** Processes camera feed for micro-expression detection.
4. **Frontend (React):** Real-time data visualization using Chart.js/D3.js.
